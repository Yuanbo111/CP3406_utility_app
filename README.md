# CP3406 Utility App

An Android utility app built for **CP3406 Mobile Computing — Assignment 1** at James Cook University.

## About

A small at-a-glance utility for a daily-life task. The current draft is an **Age & Zodiac Calculator**: pick your birth date and instantly see your age (years / months / days) and your zodiac sign.

## Tech stack

- Kotlin
- Jetpack Compose
- Material Design 3
- Android Studio

## Build & run

1. Open the project in Android Studio (Hedgehog or newer).
2. Let Gradle sync.
3. Run on an emulator or physical device.
